# SSRVM Clone

Next.js + TypeScript project for Sri Sri Ravishankar Vidya Mandir.

## Getting Started

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open http://localhost:3000 in your browser.

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run start
```

## Styling

Tailwind CSS is loaded from `src/app/globals.css`.
